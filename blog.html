Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML : innerHTML is to get or set the HTML content inside an element. It will include the HTML tags.
  • innerText: innerText is to get or set the text content of an element. It will exclude the HTML tags.
  • textContent: textContent is also to get or set thex content of an element. It will also exclude the HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is technique to attach a single event listener to a parent element instead of adding multiple event listener to child element. This technique is handly and help us write less js code. This technique works as js has concept called events Bubble Up that passes event from child to parent.

Question-4: What is event bubbling in the DOM?

When a event is triggerd in a chiled element it bubbles up through its parents, grandparents, etc. all the way up to root element. JS utilizes event bubble up for event delegation.

Question-5: How do you create, add, and remove elements using JavaScript?

  1. create element
    • document.createElement()
  2. add element
    • element.appendChild(newElement)
    • element.append(newElement)
    • element.insertBefore(newElement, element.firstChild)
  3. remove element
    • element.removeChild(childToRemove)
    • element.remove()
